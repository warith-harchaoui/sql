Langage naturel → SQL, trois approches, 100 % local (Ollama)

Exemples à cliquer

Comment ça marche ?

  • QwenCoder (brut) — on colle le schéma + la question dans un prompt maison et on appelle qwen2.5-coder. Zéro framework : tout est visible.
  • LangChain — create_sql_query_chain introspecte la base et prompte le LLM pour toi. Concis, mais plus « boîte noire ».
  • Vanna (RAG) — on « entraîne » un index (schéma + savoir métier + exemples) ; à l'exécution, seul le contexte pertinent est récupéré. Idéal gros schéma.
  • Figures — gemma4 choisit la visualisation ; on la rend en Vega-Lite. On n'exécute jamais de code produit par un LLM.

Exécution SQL en lecture seule stricte (SELECT unique, base ouverte en mode=ro). Voir PROS_CONS.md.
